API

Add new transaction

/transaction?transaction[amount]=1&transaction[description]=hello&transaction[date]=2016.08.08

Get all transactions

/transactions

Get transaction for date

/transactions/date?[date]=2016.08.03

Get total for date

/transactions/dateTotal?[date]=2016.08.03

Get total for week

/transactions/weekTotal?[year]=2016&[month]=08&[day]=06

Get total for month

/transactions/monthTotal?[year]=2016&[month]=08&[day]=06

Get total per day for month

/transactions/dailyTotalMonth?[year]=2016&[month]=08

Get expenses for date

/transactions/expenses?[year]=2016&[month]=08&[day]=06

Get incomes for date

/transactions/incomes?[year]=2016&[month]=08&[day]=06

Get transactions for range

/transactions/range?[year0]=2016&[month0]=07&[day0]=06&[year1]=2016&[month1]=08&[day1]=0

Delete transaction

/transactions/delete?[id]=1

Get monthly totals for current year

/transactions/yearsMonthTotals

Get weekly totals for current month

/transactions/monthsWeekTotals
